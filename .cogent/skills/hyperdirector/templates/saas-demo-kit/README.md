# saas-demo-kit

Product demo and feature introduction template. Two-column layout with product screenshots, feature cards, and data metrics.

## Quick Facts

| Property | Value |
|----------|-------|
| Aspect Ratio | 16:9 (9:16 supported via customization) |
| Canvas Size | 1920 × 1080px |
| Default Duration | 44s (adjustable 30–60s) |
| Scene Count | 6 (problem + reveal + 3 features + cta) |
| Motion Style | clean_tech, medium pace |

## Scene Structure

```
scene_01 [problem]         6s   dark bg, urgency, fast_scale_in
scene_02 [product_reveal]  8s   split layout + screenshot frame, scale_in
scene_03 [feature_1]       8s   left text + right card + metrics, slide_up
scene_04 [feature_2]       8s   left card + right text (mirrored), slide_up
scene_05 [feature_3]       8s   left text + right card, slide_up
scene_06 [cta]             6s   dark bg, centered, slide_up
```

## Best For

- SaaS product launch videos
- Feature announcement releases
- Internal product demo for stakeholders
- App onboarding introduction
- Website homepage video (16:9)
