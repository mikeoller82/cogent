# ai-knowledge-explainer-kit

AI concept explainer template. Dark-background, architecture-diagram aesthetic for technical knowledge sharing.

## Quick Facts

| Property | Value |
|----------|-------|
| Aspect Ratio | 9:16 |
| Canvas Size | 1080 × 1920px |
| Default Duration | 33s (adjustable 15–60s) |
| Scene Count | 5 (big_claim + context + mechanism + use_case + cta) |
| Motion Style | clean_tech, fast pace |

## Scene Structure

```
scene_01 [big_claim]   5s   dark, oversized stat, fast_scale_in
scene_02 [context]     7s   dark, card + subhead, slide_up
scene_03 [mechanism]   8s   dark, 3-node flow diagram, slide_up
scene_04 [use_case]    8s   dark, numbered step list, slide_up
scene_05 [cta]         5s   dark, button, slide_up
```

## Best For

- AI Agent workflow explanations
- Open source project introductions
- RAG / LLM / Embedding concept explainers
- Automation tutorial breakdowns
- Technical concepts for non-technical audiences
