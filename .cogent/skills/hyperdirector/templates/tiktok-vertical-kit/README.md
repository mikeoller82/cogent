# tiktok-vertical-kit

Short-form vertical video template for TikTok, WeChat Channels, YouTube Shorts, and Bilibili Story.

## Quick Facts

| Property | Value |
|----------|-------|
| Aspect Ratio | 9:16 |
| Canvas Size | 1080 × 1920px |
| Default Duration | 32s (adjustable 15–60s) |
| Scene Count | 5 (hook + 3 points + cta) |
| Motion Style | clean_tech, fast pace |
| Language | zh-CN default |

## Scene Structure

```
scene_01 [hook]     3–5s   dark bg, accent keyword, fast_scale_in
scene_02 [point_1]  6–10s  light bg, numbered card, slide_up
scene_03 [point_2]  6–10s  light bg, numbered card, slide_up
scene_04 [point_3]  6–10s  light bg, numbered card, slide_up
scene_05 [cta]      3–5s   dark bg, button, slide_up
```

| File | Purpose |

## Best For

- AI/tech knowledge sharing
- Listicle-style content ("3 things about X")
- Product feature highlights (vertical)
- Course/community promotion
- WeChat Video Channel first posts
