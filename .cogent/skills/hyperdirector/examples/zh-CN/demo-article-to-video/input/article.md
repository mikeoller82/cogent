# HyperDirector 是什么？

HyperDirector 是一个运行在 Hermes AI Agent 中的视频导演增强包。
它不是传统的视频编辑软件，而是把自然语言需求变成可渲染视频工程的自动化工作流。

## 核心能力

**第一，理解视频结构。**
它能把文章、产品页、README 自动拆解成脚本和分镜。
你不需要懂视频制作，只需要告诉它你想说什么。

**第二，记住你的品牌。**
通过 Brand Kit，它永远知道你的颜色、字体、语气和 CTA。
一次配置，所有视频自动套用。

**第三，自动质检修复。**
生成 HTML 视频后，它会自动执行 lint、检查、修复，保证渲染成功。
遇到常见错误会自动修复，最多重试 3 次。

## 适合谁用？

- AI 内容创作者：把文章、教程转成短视频，降低视频化成本
- 产品经理：把 PRD、功能截图转成 demo 视频
- 开源作者：把 README 转成项目介绍视频
- Hermes 玩家：把 Agent skill 产品化

## 最终输出

你拿到的不只是视频，而是一套完整的视频工程包：
- final.mp4：渲染完成的视频
- index.html：可编辑的 HTML 源码
- DESIGN.md：视觉设计说明
- storyboard.json：分镜结构（可直接修改）
- render-report.md：质检和渲染报告
