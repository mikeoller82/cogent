# Optional Skills for Cogent

Skills that are not activated by default but can be discovered and installed.
Browse categories below. To install a skill, use the CLI: `cogent skills install <name>`
or the /forge-skills tool in the chat.

## Categories

- **security** — Security assessment and analysis tools
- **devops** — Infrastructure, deployment, and monitoring
- **research** — Web research, OSINT, academic search
- **data-analysis** — Data processing, visualization, analytics
- **productivity** — Task management, notes, automation
- **writing** — Content creation, editing, documentation
- **creative** — Design, diagrams, media generation
