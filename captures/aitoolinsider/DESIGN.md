# AI Tool Insider — Design Reference

## Brand Colors

| Token | Hex | Usage |
|-------|-----|-------|
| `--navy` | #050D1A | Primary background (hero, main sections) |
| `--navy2` | #0A1628 | Secondary background (alternating sections) |
| `--navy3` | #0D1F36 | Tertiary background |
| `--card` | #0D1B2E | Card/panel backgrounds |
| `--border` | #1A3050 | Borders and dividers |
| `--cyan` | #00D4FF | Primary accent — CTAs, highlights, badges |
| `--blue` | #0066FF | Secondary accent — links, buttons |
| `--blue2` | #004FCC | Button hover states |
| `--white` | #FFFFFF | Body text, headings |
| `--muted` | #7A9BBF | Secondary text, descriptions |
| `--gold` | #FFD700 | Urgency badges, #1 rankings |
| `--green` | #00FF88 | Success states, positive metrics |
| `--danger` | #FF4444 | Countdown timers, limited-time badges |

## Typography

- **Font family:** `'Segoe UI', system-ui, -apple-system, sans-serif`
- **H1:** 72px, Weight 900, White
- **H2:** 42px, Weight 800, White
- **H3:** 17-22px, Weight 700-800, White
- **Body:** ~16-18px, Weight 400, #7A9BBF (muted) or #FFFFFF

## Layout

- Fixed-width centered layout (~1200px content width)
- Full-width dark background sections
- Alternating navy shades (#050D1A → #0A1628) to separate sections
- Cards with subtle border (#1A3050) on #0D1B2E backgrounds
- Spacious vertical padding between sections

## Key Design Patterns

1. **Hero:** Centered headline stack with emoji, bold CTA buttons, stats row below
2. **Magnet cards:** 4-card grid for free resource offers with hover effects
3. **Feature grid:** 6 icon-feature cards in 3×2 grid with cyan icon highlights
4. **Urgency timer:** Countdown widget in red/gold with "THIS WEEK'S WINDOW" label
5. **Tool cards:** Product cards with rating stars, category badges, price tag, CTA
6. **Comparison table:** 3-column table (AIToolInsider vs Generic Blog vs Random Newsletter)
7. **Testimonial cards:** Star rating + quote + author attribution
8. **Pricing card:** Feature list with checkmarks, bold CTA

## Video Direction Notes

- Dark background = glowing cyan accent elements will pop beautifully in motion
- Use cyan (#00D4FF) as the primary motion accent, gold (#FFD700) for urgency moments
- Keep text large and readable — the hero headline at 72px is the hook
- Emphasize the "tested, not hyped" positioning — credibility is the core brand promise
- Motion style: clean, technical, fast-paced — matches the "stay ahead" energy
