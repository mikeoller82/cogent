# /ponytail — Ponytail Mode Control

Sets the active ponytail intensity for the current session.

## Syntax

```
/ponytail          → show current mode
/ponytail off      → disable ponytail rules
/ponytail lite     → light YAGNI nudge, no strict rung ladder
/ponytail full     → full lazy-senior-dev ladder (default)
/ponytail ultra    → maximum deletion pressure, zero tolerance for abstraction
```

## Modes

| Mode   | Effect |
|--------|--------|
| off    | Standard Cogent behavior — no ponytail rules applied |
| lite   | Soft YAGNI reminder, no strict rung ladder |
| full   | All 7 rungs enforced: YAGNI → codebase reuse → stdlib → native → deps → one-liner → minimum code |
| ultra  | Same as full, plus aggressive deletion. Every file must justify its existence |

When the mode changes, the system prompt is updated on the next turn.
